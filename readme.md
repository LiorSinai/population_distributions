## Set up (optional)

Create Conda environment:

```bash
conda create --name pygeo --python=3.9
```

Install packages in the requirements.txt file.

Set up Conda Kernel:
```bash
conda activate pygeo
python -m ipykernel install --user --name=pygeo
conda activate pygeo
jupyter notebook
```

Use with Jupyter Notebook:

```bash
conda activate pygeo
jupyter notebook
```
